# Project Title

Backend Hotel CRUD With Express js, Node Js, And Sequelize

## Run Locally

Clone the project

```bash
  git clone https://github.com/Abyansyah/BE_Hotel_UKL.git
```

Go to the project directory

```bash
  cd BE_Hotel_UKL
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  nodemon
```

## Build With

- NodeJs
- Express
